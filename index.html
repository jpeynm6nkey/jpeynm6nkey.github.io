Sistema de Detección de Phishing con Machine Learning

Desarrollo de un modelo de IA para detectar sitios web de phishing a partir del análisis de URLs. Incluyó preprocesamiento de datos reales, balanceo del dataset, extracción de variables relevantes y entrenamiento con distintos algoritmos.

  • Análisis de ventajas y desventajas del uso de IA para la detección de sitios web fraudulentos.
  • Preprocesamiento de datos desde Kaggle (>500K URLs).
  • Transformación y limpieza de los datos para su uso en modelos de Machine Learning.
  • Balanceo del dataset para corregir el desbalance entre clases (good/bad).
  • Extracción de variables (Recuento de símbolos, de anclas, de puntos en la URL, presencia de SSL).
  • Algoritmos: MLPClassifier, Naive Bayes, SVM, Árboles de decisión.
  • Métricas: Accuracy 65%, Precision 73%, Recall: 46%, F1 57%, Curva ROC y Matriz de Confusión.
  • Gráficas: Seaborn, Matplotlib.
  • Discusión de desafíos éticos y técnicos (Necesidad de datos de calidad; Costes de mantenimiento y entrenamiento; Falsos positivos y negativos
  • Ética y normativa RGPD.
